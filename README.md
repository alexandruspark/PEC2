# PEC2
Formula 1 pit-stop checker depending on remaining fuel laps, tire type, tire wear and whether a safety car is deployed or not in C

Example:
INPUT
FUEL LAPS REMAINING?
10
TIRE TYPE (0-SOFT, 1-MEDIUM, 2-HARD)?
1
TIRE WEAR PERCENTAGE (0-100)?
82
IS SAFETY CAR DEPLOYED (0-FALSE, 1-TRUE)?
0
OUTPUT
PIT STOP RECOMMENDED (0-FALSE, 1-TRUE): 1
7
